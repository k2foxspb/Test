Windows
    Установка:
        скачиваем проект, устанавливаем виртуальное окружение python -m venv venv:
            venv\Scripts\activate.bat
        в виртуальном окружении:
            pip install -r requirements.txt
        или в глобальном окружении:
            pip install pyserial
    Запуск:
        в папке проекта:
            python launcher.py

    Переменные в var.py

    Возможные ошибки:
        в файле launcher изменить ('py m1.py', 'py m2.py', 'py m3.py') на ('python m1.py', 'python m2.py' 'python m3.py')

Проблемы:
    если отправить кодированные данные 16-тиричной записи, то на выходе выдаёт кодировку в попугаях
